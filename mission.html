Mission

Dans le cadre de mon BTS SIO SLAM, j’ai développé une application de gestion des absences inspirée de l’application “Habilitations”. Cette application PHP/MVC permet la gestion du personnel et des absences via une interface administrateur.

  • Connexion sécurisée
  • Gestion du personnel (ajouter, modifier, supprimer)
  • Gestion des absences (ajouter, modifier, supprimer, consulter)
  • Base de données MySQL + architecture MVC
